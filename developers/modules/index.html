The CitySDK has a modular structure. The core library provides functions that can be used by various modules. A module in the CitySDK context provides the connection to various data sources. For example, the census module provides connection and processing of census-related APIs and data.

The CitySDK Core

citysdk.js

This library provides communications functions, the caching system for CitySDK, and a few hard-coded data sets (such as a directory of state capitals and their latitude and longitude). Individual modules are responsible for deciding how to use the caching system and when to clear the cache.

ArcGIS

citysdk.arcgis.js

This module allows the CitySDK to retrieve data from ArcGIS servers including lists of services, layer/level descriptions, and data from individual levels of a FeatureService.

Census

citysdk.census.js

The census module provides functionality to query various census APIs. There are three primary API groups that can be used:

GeoCoder

This is access via a set of functions that convert various locations (address, zip, coordinates, etc) into FIPS codes which are used by the other census APIs.

TigerWeb

This is the Census mapping service that provides shape files that can be overlaid onto maps. It is accessed via the GEORequest() function.

Api.census.gov

Api.census.gov - These are the various surveys that are available from the Census. It is accessed via the APIRequest() and/or GEORequest() function.

Key is Required. If you don't already have one, be sure to grab a Census API key. Get a Key

Energy Information Agency (EIA)

citysdk.eia.js

The EIA module provides access to various summary reports from the Department of Energy. It returns lists of information categories and individual series of data.

Creating New Modules

Don't see what you need? If you need data that isn't provided by one of these modules, you have a couple options. You can build your own module and submit a pull request or make a request on the CitySDK Github to have a new module created.

For more information on how to build a new module, see the guide: How to Create a New CitySDK Module.
